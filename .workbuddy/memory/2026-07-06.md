# 2026-07-06

## 新增"接收分享贴图"功能

**变更内容**：
1. 新建 `lib/SharePasteHelper.java` — 工具类，封装接收分享的核心逻辑：
   - `handleShareIntent(Activity, Intent)` — 入口调度，区分 ACTION_SEND / SEND_MULTIPLE
   - `floatText(Activity, String)` / `showFloatText(Activity, String)` — 文字贴图，含权限检查
   - `pasteImage(Activity, Uri)` / `showImageFloat(Activity, Bitmap)` — 图片贴图，从 Uri 加载 Bitmap
   - `loadBitmapFromUri(Context, Uri)` — 通用 Uri → Bitmap 转换
   - 维护静态列表 `helperFloatTags` / `helperImageTags` 供 HomeFragment 透过度量/清理

2. 新建 `ShareReceiverActivity.java` — 透明 Activity，无 UI，接收第三方分享 Intent 后立即委托给 SharePasteHelper 处理并 finish

3. 修改 `AndroidManifest.xml` — 注册 ShareReceiverActivity，添加三个 intent-filter：
   - `ACTION_SEND` + `text/plain`
   - `ACTION_SEND` + `image/*`
   - `ACTION_SEND_MULTIPLE` + `image/*`

4. 修改 `res/values/theme.xml` — 新增 `Theme.Transparent` 透明主题

5. 修改 `fragment/HomeFragment.java` — `setFloatViewOpacity()` 和 `clearAllTextFloatViews()` 增加对 SharePasteHelper 静态列表的遍历，使共享贴图也受透明度滑块和清空按钮控制
